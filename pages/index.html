部署方式

GitHub Pages

当前页面是静态项目主页，适合展示说明和部署文档。GitHub Pages 不能运行 Express、SQLite 或 RAGFlow。

云服务器

真正的应用需要用 Docker Compose 部署在云服务器上，并接入同机或内网 RAGFlow。

docker compose up -d --build
